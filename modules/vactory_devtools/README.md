# Vactory devtools
Enable all necessary dev modules, this is a non production contrib module.

### Installation
`drush en vactory_devtools -y`

### Dependencies
- devel
- devel_generate
- twig_vardumper
- webprofiler
- modules_weight
- security_review
- views_ui
- features_ui
- field_ui
- stage_file_proxy
- module_filter
- styleguide
